# Rust Articles — Curated

A short curated list of insightful articles and long-reads useful when learning Rust.

- **The Rustonomicon** — Deep dive into Rust's unsafe features and advanced topics. https://doc.rust-lang.org/nomicon/
- **The Rust Programming Language (book)** — Comprehensive, official guide to Rust. https://doc.rust-lang.org/book/
- **Rust by Example** — Learn Rust through runnable examples. https://doc.rust-lang.org/rust-by-example/
- **Learning Rust With Entirely Too Many Linked Lists** — A progressive tutorial showing ownership and borrowing. https://rust-unofficial.github.io/too-many-lists/
- **Fearless Concurrency in Rust** — Articles and posts explaining Rust's concurrency model and safety guarantees. https://doc.rust-lang.org/book/ch16-00-concurrency.html
- **Understanding Ownership, Borrowing, and Lifetimes** — Guides that explain core Rust concepts (search for blog posts by Amos and Jonathan Turner).
- **Error handling in Rust** — Best practices for `Result`, `Option`, and error propagation (`?`). https://doc.rust-lang.org/book/ch09-00-error-handling.html
- **Async in Rust: a primer** — Introductions to `async/.await`, executors, and popular crates like `tokio` and `async-std`.
- **Cargo and Crates.io: ecosystem guide** — How to publish, manage dependencies, and version crates.
- **Rust for Systems Programmers** — Articles exploring how Rust replaces C/C++ for systems-level work (various blog posts from AWS, Mozilla, and community blogs).
